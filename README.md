## Features

- Luxury dark aesthetic with gold accents
- Custom cursor with lag effect
- 3D tilt on stats card and feature cards
- Scroll reveal animations
- Animated marquee strip
- Menu filter tabs
- Auto-scroll review slider
- Draggable review slider
- Count-up number animations
- Pulsing CTA rings
- Floating background orbs
- Scroll progress bar
- Mobile hamburger menu
- Fully responsive (mobile + desktop)
- WhatsApp direct order link
- Semantic HTML + accessibility

---

## How to Open

Simply open `index.html` in any modern browser.  
No build step, no dependencies, no server needed.
